# Blood Bank Management System

A simple C++ console application to manage donor and volunteer registrations for a blood bank.  
The system allows users to register, an admin to view records and handle emergency blood requests.  
All data is saved into `bloodbank.txt` for persistence

# Features
- Register as Donor
- Register as Volunteer
- Admin login with password "1234"
- Emergency blood request
- compatibility check
- View all donors and volunteers
- File management
